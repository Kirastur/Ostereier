What's new
* Integration to the Citizens-Plugin: You can open the shop by clicking a NPC
* Sanity Scheduler for periodical sanity checks
* Improved Admin Notifications

Configuration
* New parameter "startup/initialSanityCheckDelay"
* New section "sanity" for periodical sanity checks
* new section "npc" for Citizens integration
 
Changes in API
* The functions for starting and stopping the editor and game now have an additional parameter "initiator" 
* The function "sanityCheck" was renamed to "performSanityCheck"

Messages
Some messages have changed due to the reorganization of the admin notifications

Please delete your old config.yml and messages.yml, so the plugin can create a fresh one including the new parameters.
